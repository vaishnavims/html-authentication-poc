# Styled Authentication System

This project is a **basic authentication user interface** built with **HTML, Bootstrap 5, and custom CSS**.
It demonstrates a simple authentication flow including login, registration, password recovery, and a dashboard page.

The goal of this project is to convert plain HTML authentication pages into a **responsive and visually improved interface using Bootstrap components and custom styling.**

---

## Pages

The project includes the following pages:

* **login.html** – User login page
* **register.html** – New user registration page
* **forgot-password.html** – Request password reset
* **reset-password.html** – Set a new password
* **dashboard.html** – Basic user dashboard after login

---

## Technologies Used

* HTML5
* Bootstrap 5
* Bootstrap Icons
* Custom CSS
* Google Fonts

---

## Key Features

* Responsive design compatible with desktop, tablet, and mobile screens
* Bootstrap components such as cards, forms, buttons, and navbar
* Custom styling with gradients and shadows
* Password visibility toggle using icons
* Password strength indicator during registration
* Simple navigation between authentication pages
