# Simple Authentication Pages

This project contains basic HTML pages for a simple authentication flow.

## Pages Included

- login.html
- register.html
- forgot-password.html
- reset-password.html
- dashboard.html

## Description

The pages are connected using anchor tags to simulate navigation between:

- Login
- Registration
- Forgot Password
- Reset Password
- Dashboard

There is no CSS, JavaScript, or backend functionality included.
This project focuses only on basic HTML structure and page linking.

## How to Use

1. Open `login.html` in a browser.
2. Use the links to navigate between pages.
